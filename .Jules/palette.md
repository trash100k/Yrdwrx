## 2025-05-14 - Fulfilling UI Promises for Keyboard Navigation
**Learning:** When a UI component explicitly mentions keyboard shortcuts (e.g., "Arrow keys to navigate", "Enter to select"), it is critical that these interactions are actually implemented. Discrepancies between UI instructions and actual behavior lead to user frustration and poor accessibility.
**Action:** Always verify that "power user" instructions in the UI are backed by functional keyboard listeners and visual state updates.
